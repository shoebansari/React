Typescript is open-source language developed by Anders Hejlberg at Microsoft.It is superset of javascript that compiles to plain javascript. it run on any browser, operating system, that means all valid javascript code is also typescript code. It offers advance feature such as Intellisense, code compilation, safe refactoring etc. JavaScript is a loosely typed language. It can be difficult to understand what types of data are being passed around in JavaScript. In JavaScript, function parameters and variables don't have any information! So developers need to look at documentation, or guess based on the implementation.
